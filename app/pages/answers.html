1.Что такое EcmaScript?

EcmaScript-скриптовый язык программирования,созданый для работы с существующим объектом или системой.Стандарт языка JavaScript.

2.Для чего используются и чем отличаются атрибуты async и defer?

Позволяет загружать скрипт в фоновом режиме без блокировки загрузки остального контента. Отличие атрибутов: async-скрипт выполнится, как только будет загружен,а defer-скрипт будет выполнен после полной загрузки страницы и сохраняется очередность выполнения.

3.Почему предпочтительнее подключать скрипты перед , а не в ...?

При подключении скрипта в , скрипт загрузиться после загрузки всей страницы, а в ... скрипт может не загрузится, потому что не подгрузится страница с html-документом.

4.Какие основные отличия интерпретируемого языка программирования от компилируемого?

Компилируемый язык-это такой язык,что программа,будучи скомпилированной, содержит инструкции целевой машины; машинный код непонятен людям. Интерпретируемый язык-это такой, в котором инструкции не исполняются целевой машиной,а считываются и исполняются другой программой. Главное отличие компилируемого языка скорость исполнения,они конвертируются в машинный код, работают гораздо быстрее и эффективнее нежели интерпретируемые. Кроме того,компилируемые языки дают разработчику больше возможностей в плане контроля аппаратного обеспечения, например, управление памятью и использованием процессора. Интерпретируемом языке для исполнения программы не нужен машинный код; вместо этого программу построчно исполняет интерпретаторы. Раньше процесс интерпритации занимал много времени, но с приходом таких технологий, как JIT-компиляция, разрыв между языками сокрощается.

5.Что такое языки высокого и низкого уровня? Какие ты знаешь?

Низкоуровневый язык пограммирования, близкий к программированию непосредственно в машинных кодах используемого реального или виртуального процессора. Низкоуровневые языки: C, C++, Haskell, Rust и Go. Высокоуровневый язык программирования разработанный для быстроты и удобства использования программистом. Основная черта высокоуровневых языков введение смысловых конструкций, кратко описывающих такие структуры данных и операции над ними, описания которых на машинном коде очень длинны и сложны для понимания. Высокоуровневые языки: JavaScript, PHP, Ruby, Python.

6.Для чего используются переменные? Какие ключевые слова используются для создания переменных? Какие есть отличия в переопределении значений при задании переменной через let, const и var?

Чтобы хранить информацию. Для создания переменных используются ключивые слова var,let, const. Переменную,заданную через var,легко переопределить,а let и const прямо переопределить нельзя.

7.Может ли имя переменной начинаться с цифры, символов _ или $

Имя переменной может начинаться с символов _ и $, а цифры не может.

8.Какие слова запрещено использовать в названии переменной?

запрещено использовать для пременных зарезервированные слова самим языком. Например: var, class, return,export.

9.Что такое camelCase и snake_case?

CamelCase стиль написания составных слов, при котором несколько слов пишутся слитно без пробелов, при этом каждое слово внутри фразы пишется с прописной буквы. Snake_case стиль написания составных слов, при котором несколько слов разделяются символом (_), не имеют пробелов в записи, причем каждое слово обычно пишется с маленькой буквы.

10.Что такое "типы данных"? Какой оператор позволяет определить тип данных?

Тип данных способ хранения информации. 6 типов данных: число(number),строка (string),лочисеский (boolean), символ (symbol), отсуствие какого-то значения(null), значение не задано (undefined) и объект(objects). Оператор typeof определяет тип данных.

11.Что такое "динамическое приведение типов данных"? Можно ли писать JavaScript со статическим приведением типов?

Динамическое приведение типов данных-это значит что не нужно определять тип переменной зарание. Тип определиться автоматически во время выполнения программы. Не можно JavaScript язык с динамической типизацией.

12.Как привести строчное значение "10" к числу? Назовите два способа.

Для преобразования к числу в явном виде можно вызвать Number(),либо поставить перед выражением унарный плюс"+". При помощи выражения ${...}. Например: let apples = 2; let oranges = 3; alert (`${apples} + $(oranges) = ${apples + oranges}`);// 2+3=5

13.Чем отличаются операторы "==" и "==="?

Оператор "==" сравнивает значения,а оператор "===" сравнивает значения и тип данных.

14.Какие преимущества дают обратные кавычки (`) при указании строки?

Возможен перевод строки с пробелами. Можно вставлять выражения при помощи ${...}.
